Imagyy.

Due to the advent of Post Search, it's difficult to use the feature of "Photos of" which was earlier done using graph search. This application aims to solve that problem. Currently it supports the following social networking platforms.

  • Facebook
  • Instagram
  • LinkedIn
  • Github
  • Quora
